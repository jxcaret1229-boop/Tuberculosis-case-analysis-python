# TB-data-analysis-python

This project focuses on the analysis of Tuberculosis (TB) data using Python. It includes data cleaning, manipulation, and exploratory analysis to better understand the dataset and identify useful insights.

## Project Sections

- **Tuberculosis Analysis**  
- **Data Cleaning and Manipulation**  
- **Exploring Data with Python**  

To practice Python data analysis skills by working with a real dataset and presenting results through exploration and visualization.
